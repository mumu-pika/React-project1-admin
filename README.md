### 简介
这是一个使用React框架搭建的后台管理系统（前端界面）。
后端使用的是一个本地Node + Express + Mongodb 的数据库接口获取数据。
(调用接口的实践)
学习测试用，由于后台数据库数据丢失，等之后考虑springboot等技术搭建后台。

### 技术栈
React + Antd + Axio + Bizcharts + Webpack打包等技术。
模块化 + 工程化 + 组件化的模式开发。
跨域的解决方案：采用服务器中转代理，使用http-proxy-middleware中间件处理。
